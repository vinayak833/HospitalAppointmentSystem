# Hospital Appointment System

A web-based Hospital Appointment System built using HTML, CSS, and JavaScript. The application helps manage doctors, patients, and appointments through a simple and user-friendly interface with Local Storage support.

## Features

### Doctor Management
- Add doctors
- Store doctor specialization
- View doctor list
- Manage doctor records

### Patient Management
- Add patients
- Store patient details
- View patient records
- Manage patient information

### Appointment Booking
- Book appointments
- Select doctor and patient
- Choose appointment date
- View scheduled appointments

### Dashboard
- Total Doctors Count
- Total Patients Count
- Total Appointments Count
- Real-time statistics

### Local Storage
- Save doctors data
- Save patients data
- Save appointment records
- Automatically restore data after page refresh

### User Interface
- Clean layout
- Responsive design
- Easy navigation
- Beginner-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Hospital-Appointment-System.git
```

2. Navigate to the project directory

```bash
cd Hospital-Appointment-System
```

3. Open `index.html` in your browser.

## How It Works

### Adding Doctors
1. Enter doctor name.
2. Enter specialization.
3. Click "Add Doctor".

### Adding Patients
1. Enter patient name.
2. Enter patient age.
3. Click "Add Patient".

### Booking Appointments
1. Select a doctor.
2. Select a patient.
3. Choose a date.
4. Click "Book Appointment".

### Dashboard
- Displays total doctors.
- Displays total patients.
- Displays total appointments.
